Información sobre el Proyecto

Propósito del Proyecto

El propósito principal de este proyecto es desarrollar una aplicación web sencilla y eficiente para ayudar a los usuarios a gestionar sus tareas diarias de manera fácil e intuitiva. Nuestra meta es mejorar la productividad y la organización de las personas, brindándoles una herramienta que les permita mantener un control efectivo de sus actividades.

Tecnologías Utilizadas

  • HTML, CSS, Bootstrap: Utilizados para el desarrollo del frontend de la aplicación, brindando una interfaz de usuario responsive y de fácil uso.
  • Vercel: Plataforma de despliegue en la nube que nos permite publicar la aplicación de manera sencilla y accesible en línea.
  • Docker, Docker Play: Herramientas de containerización que facilitan la ejecución y distribución de la aplicación en diferentes entornos.
  • Git, GitHub: Sistema de control de versiones que nos permite colaborar de manera eficiente en el desarrollo del proyecto.
