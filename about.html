Información sobre el Proyecto

Propósito del Proyecto

El propósito de este proyecto es desarrollar una aplicación web sencilla para gestionar una lista de tareas de manera fácil e intuitiva. Esto ayudará a los usuarios a organizar y mantener un control efectivo de sus actividades diarias.

Tecnologías Utilizadas

  • HTML, CSS, Bootstrap
  • Vercel (Despliegue)
  • Docker, Docker Play (Contenedores)
  • Git, GitHub (Control de Versiones)
